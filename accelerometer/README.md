## Code to control DFRobot Accelerometer.

### Links

https://wiki.dfrobot.com/Gravity_I2C_LIS2DW12_Triple_Axis_Accelerometer_SKU_SEN0409#target_6\
https://github.com/DFRobot/DFRobot_LIS

### Connections

* '+' => 17 (3.3V pin)
* '-' => 6 (GND)
* C => 3
* D => 5

Configuration pin towards 0x19

### Requirements

* build-essential
* python3-dev
* pytho3n-smbus
* git

### Instructions

* sudo chmod a+w x.py
* python3 dir/file.py
